🎵 MelosFree: 저작권 프리 음악 스트리밍 웹앱
MelosFree는 크리에이터, 코더, 그리고 공부하는 사람들을 위해 저작권 걱정 없는 음악을 24시간 제공하는 무료 스트리밍 웹 애플리케이션입니다.

🚀 프로젝트 개요
저작권 침해 우려가 없는 Royalty-Free 음원을 큐레이션하여 사용자에게 제공합니다. 복잡한 로그인 없이 접속 즉시 음악을 들을 수 있으며, 반응형 설계를 통해 모바일과 데스크톱 어디서든 백그라운드 재생이 가능합니다.

✨ 핵심 기능
Copyright-Free Library: YouTube 오디오 라이브러리, Bensound 등 신뢰할 수 있는 소스의 음원만 제공.
Seamless Background Playback: 페이지 이동 시에도 음악이 끊기지 않는 재생 환경.
Simple UI/UX: 최소한의 조작으로 원하는 장르의 음악을 탐색하고 재생.
PWA(Progressive Web App) 지원: 앱처럼 설치하여 오프라인 캐싱 및 백그라운드 제어 가능.
Visualizer: 음악 비트에 맞춰 반응하는 실시간 오디오 시각화 도구.

🛠 기술 스택
Frontend: React.js+vite
State Management: Zustand 또는 React Context API (재생 상태 관리)
Styling: Tailwind CSS (빠르고 가벼운 UI 구현)
Audio Engine: Howler.js (크로스 브라우저 오디오 호환성 확보)
Hosting: 미정
